Button

Assignment

Write an HTML file that has Javascript code in it. The filename will be called button.html

The JS code and HTML code will do the following:

  1. Create an HTML button that will trigger the execution of a function named doMath
  2. The doMath function will prompt the user for three numbers and will determine which number is the smallest.
  3. The function will print out to the user via an alert of the smallest number.
